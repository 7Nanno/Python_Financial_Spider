#项目简介
这是一个专门用来抓取国内实时金价（AU9999）的小工具。原本是想练习爬虫，后来发现金融数据接口比普通网页难搞得多，所以专门立了这个项。
#突破的技术难点（必看）
我在开发过程中发现，直接请求接口会被拒绝，后来我通过研究发现并解决了以下问题：
* **Referer 防盗链**: 模拟了新浪财经的来源地址，才拿到了数据权限。
* **字符编码转换**: 解决了接口回传的 GBK 乱码问题，确保价格显示正常。
* **数据解析**: 采用“剥洋葱”法，从复杂的字符串中精准提取出了实时价格。
### 当前功能
* 实现了每 10 秒自动获取一次上海金交所实时金价。
* 自动计算并输出“元/克”的价格。
